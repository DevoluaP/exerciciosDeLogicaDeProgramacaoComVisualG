## Sobre o Repositório

Este repositório reúne exercícios desenvolvidos em **Visualg**, com foco no aprendizado e prática de **lógica de programação**.

Os algoritmos foram criados ao longo dos estudos e cobrem desde conceitos básicos até problemas um pouco mais avançados.

---

## Observação

**Nenhum arquivo possui acentuação (á, é, ç, etc.)**
Isso foi feito propositalmente para evitar conflitos de compatibilidade no Visualg.

## Exemplos de Algoritmos

### Bubble Sort

```alg
Var
   numeros: vetor[1..5] de real
   ordem, indice: inteiro
   valorTemporario: real

Inicio
   escreval("Ordenacao por Bubble Sort")

   para ordem <- 1 ate 5 faca
      escreva("Digite o ", ordem, "° numero: ")
      leia(numeros[ordem])
   fimpara

   para ordem <- 1 ate 4 faca
      para indice <- 1 ate 5 - ordem faca
         se numeros[indice] > numeros[indice+1] entao
            valorTemporario <- numeros[indice]
            numeros[indice] <- numeros[indice+1]
            numeros[indice+1] <- valorTemporario
         fimse
      fimpara
   fimpara

   escreval("")
   escreval("Numeros em ordem crescente:")
   para ordem <- 1 ate 5 faca
      escreval(numeros[ordem]:8:2)
   fimpara

Fimalgoritmo
```

### Palíndromo Numérico

```alg
Var
   numero, verifica, invertido, digito: inteiro

Inicio
   escreval("Digite um numero inteiro positivo:")
   leia(numero)

   verifica <- numero
   invertido <- 0

   enquanto verifica > 0 faca
      digito <- verifica mod 10
      invertido <- (invertido * 10) + digito
      verifica <- verifica div 10
   fimenquanto

   se numero = invertido entao
      escreval(numero, " e um palindromo.")
   senao
      escreval(numero, " nao e um palindromo.")
      escreval("Invertido: ", invertido)
   fimse

Fimalgoritmo
```
